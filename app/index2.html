List of ES6 features:

  1. New ES6 syntax
    • let – Declare block-scoped variables using the let keyword.
    • let vs. var vs. const – Differences between let, var and connst.
    • Default function parameters – Set the default value for parameters of a function.
    • Rest parameter – Rest parameter and how to use them effectively.
    • Spread operator – How to use the spread operator effectively.
    • Object literal syntax extensions – Provide a new way to define object literal.
    • for…of – How to use the for...of loop to iterate over elements of an iterable object.
    • Octal and binary literals – Support for binary literals and change the way to represent octal literals.
    • Template literals – Substitute variables in a string.
  2. ES6 Modules
    • ES6 modules – Write modular JavaScript code.
  3. ES6 Classes
    • Class – The ES6 class syntax and how to declare a class.
    • Getters and Setters – Define the getters and setters for a class using the get and set keywords.
    • Class Expression – An alternative way to define a new class using a class expression.
    • Static methods – How to define methods associated with a class, not instances of that class.
    • Static Properties – Define static properties shared by all instances of a class.
    • Computed property – The computed property and its practical application.
    • Inheritance – How to extend a class using the extends and super keywords.
    • new.target – The new.target meta property.
  4. Arrow Functions
    • Arrow functions –The arrow functions ( => )
  5. Destructuring
    • Array Destructuring – How to assign elements of an array to variables.
    • Object Destructuring – How to assign properties of an object to variables.
  6. Iterators & Generators
    • iteration and iterator protocols.
    • Functions that can pause midway and then continuefrom where they paused.
    • yield – How to use the yield keyword in generators.
  7. Promises
    • Promises – Javascript Promises.
    • Promise chaining – How to execute multiple asynchronous operations in sequence.
    • Promise composition: Promise.all() & Promise.race() – How to compose a new promise out of several promises.
    • Promise error handling – How to handle errors in promises.
  8. Array Extensions
    • Array.of() – Improve array creation.
    • Array.from() – Create arrays from array-like or iterable objects.
    • Array find() – Find an element in an array
    • Array findIndex() – Find the index of an element in an array.
  9. Object extensions
    • Object.assign() – Copy an object or merge objects.
    • Object.is() – Check if two values are the same value.
  10. String Extensions
    • String startsWith() – Check if a string starts with another string.
    • String endsWith() – Determine if a string ends with another string.
    • String includes() – Check if a string contains another string.
  11. Symbol
    • Symbol – A new primitive type called symbol in ES6.
  12. Collections
    • Map – The Map type that holds a collection of key-value pairs.
    • Set – How to use the Set type that holds a collection of unique values.
  13. Proxy & Reflection
    • Proxy – How to use the proxy object that wraps another target object and intercepts the fundamental operations of the target object.
    • Reflection API – How to use ES6 Reflection API to manipulate variables, properties, and methods of objects at runtime.
